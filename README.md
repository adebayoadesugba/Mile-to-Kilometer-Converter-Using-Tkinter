# Miles to Kilometers Converter (Python Tkinter)

A simple **Graphical User Interface (GUI) application** built with **Python Tkinter** that converts **miles to kilometers**.

The user enters a value in miles, clicks the **Calculate** button, and the program instantly displays the equivalent distance in **kilometers**.

This project demonstrates **GUI development with Tkinter, event-driven programming, and basic mathematical conversion in Python**.

---

## Features

- Simple and user-friendly graphical interface
- Converts miles to kilometers instantly
- Interactive button to perform calculation
- Displays result dynamically
- Built using Python's built-in **Tkinter GUI framework**

---

## Technologies Used

- Python 3
- Tkinter (Python GUI library)
- Event-driven programming

## How to Use

1. Enter a distance value in **Miles**.
2. Click the **Calculate** button.
3. The equivalent **Kilometers** value will be displayed.

Example:

```
Input: 5 Miles
Output: 8.0 Kilometers
```

Conversion Formula:

```
1 Mile = 1.6 Kilometers
```
